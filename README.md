# Clinical SAS SDTM DM Domain Creation

This project demonstrates the creation of an SDTM Demographics (DM) domain using SAS.

## Workflow

1. Import raw clinical data from Excel.
2. Create USUBJID according to CDISC standard (STUDYID-SITEID-SUBJID).
3. Convert dates to ISO8601 format.
4. Merge Exposure and Randomization datasets.
5. Create treatment arm variables.
6. Generate SDTM DM dataset.
7. Export dataset to XPT format.
8. Validate using Pinnacle21 Community.

## Tools Used

- SAS Programming
- CDISC SDTM Standards
- Pinnacle21 Validator
- Git & GitHub

## Output

- SDTM DM Dataset
- XPT Transport File
- Pinnacle21 Validation Report
